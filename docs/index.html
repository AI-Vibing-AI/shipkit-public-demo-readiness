Need the full demo shell?

The optional full kit adds ready-made pause, settings, input remap, reset confirmation, feedback, store CTA, controller-label, and build-stamp UI for Godot 4.2 through 4.6.x.

Download it free if you need to inspect fit first. Paid/supporter downloads are optional and help fund install docs, compatibility smoke tests, and small Godot compatibility fixes.

Boundary:

This free checklist does not include the full shell's pause menu, settings menu, input remapping, controller labels, reset confirmation UI, feedback button, or store CTA button. You do not need the full shell to use the free checklist.

Pick the right path

Use the free checklist if

  • your project already has its own menu and settings flow
  • you only need a readiness review before a public build
  • you want a tiny MIT inspector you can copy apart

Open the full shell if

  • you still need pause, settings, input remap, reset, feedback, and store CTA UI
  • you want a wrapper you can adapt instead of rebuilding those pieces
  • a supporter download would be worth less than the setup time it saves

Install sketch

  1. Download and unzip the free release.
  2. Copy `addons/shipkit_public_demo_readiness` into a Godot 4 project.
  3. Enable the plugin, or instance the inspector scene in a debug/QA scene.
  4. Run the checklist before sending the demo to public players.
